# Data-structure 02
## Base64 인코딩
- btoa() 함수는 문자열로부터 Base64 인코딩된 ASCII 문자열을 생성
- 문자열의 각 문자는 바이트로 취급
- atob() 함수는 Base64 인코딩을 사용해 인코딩된 자료의 문자열을 디코딩

## 문자열 단축 알고리즘
- Bit.ly와 같은 단축 URL을 가진 사이트는 특정 알고리즘을 가짐
  1. 데이터베이스가 URL에 대해서 정수에 기반한 고유 ID를 생성
  2. 정수 ID를 Base64인코딩을 사용하여 문자열로 단축
